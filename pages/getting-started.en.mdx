## Getting started with the pier SDK and Dashboard

Before you integrate the pier SDK, you need to obtain an API Key.

1. Go to https://dashboard.pierwallet.com
2. Sign up or Log in
3. Go to the "Developers" tab
4. Create the API Key & Store it somewhere safe
