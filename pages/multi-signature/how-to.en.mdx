## What you can do

Go to the [pier multisig interface](https://multisig.pierwallet.com) where you can:

- [x] View & Edit wallet settings (e.g. name, description, logo)
- [x] View & Propose to edit signatory wallets (if you have the correct rights)
- [x] View & Propose add / remove whitelist for transactions
- [x] View & Confirm / reject transaction proposals
- [x] View historic transactions
- [x] Export history / audit trail
