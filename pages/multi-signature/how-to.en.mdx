## What you can do

Go to the [pier multisig interface](https://dashboard.pierwallet.com) where you can:

- [x] Edit project settings (e.g. name, description, logo)
- [x] Propose to add / remove user wallets

- [x] Propose to add / remove guardian wallets
- [x] Propose add / remove whitelist for transactions
- [x] Confirm / reject proposals

- [x] Export history / audit trail
