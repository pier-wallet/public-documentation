# Wallet as a Service with the simplicity of Web2

pier offers it's services in two different models:

- Account Abstraction SDK: pier SDK is a set of functions that you can integrate into your application. It gives you access to a fully managed service, meaning that you don't have to worry about the underlying infrastructure. You can use pier SDK to create wallets, sign transactions and interact with the blockchain.

- Multi Party Computing (MPC) SDK: pier MPC SDK gives you access to all of pier's MPC capabilities - you can create wallets, sign transactions and interact with the blockchain. pier MPC SDK is a fully managed service, meaning that you don't have to worry about the underlying infrastructure.

- White Label: pier White Label is a fully managed service that allows you to create wallets, sign transactions and interact with the blockchain. It is a multi-tenant platform, meaning that multiple customers can use the same platform, but their data is isolated from each other. pier White Label is a fully managed service, meaning that you don't have to worry about the underlying infrastructure.
